# AI Burn English — AI共燃英語團

> 21 Days. A group of people whose English isn't great. Getting better together with AI.

## What is this?

The **AI Burn English** website — the official home of a 21-day AI-assisted English co-learning group. Not for people who are already good at English, but for people who want to get better — together.

## Tech Stack

- **Runtime**: Bun
- **Framework**: Astro 5 (static output)
- **Styling**: Tailwind CSS v4
- **Deployment**: Cloudflare Pages

## Local Development

```bash
# Install dependencies
bun install

# Start dev server
bun dev

# Build for production
bun run build
```

## Deployment

Push to `main` → GitHub Actions deploys to Cloudflare Pages automatically.

## Project Structure

```
src/
├── components/    # Astro components
├── layouts/      # Page layouts
├── pages/        # Routes
├── content/      # MDX posts
├── styles/       # Global CSS
└── lib/          # Constants
```

## Philosophy

- Warm, handcrafted feel — not cold AI aesthetic
- Paper texture + fire palette
- Mobile-first, accessible
- Written in Traditional Chinese (zh-TW) first
